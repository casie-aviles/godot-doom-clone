# Godot Doom Clone

A DOOM-style FPS made with Godot 4.

## Quick Start

Open `doom-clone/project.godot` in Godot 4.x and run.

## Controls

- **WASD** - Move
- **Mouse** - Look
- **Left Click** - Shoot
- **Right Click** - Kick
- **Shift** - Run
